### üéµ [FullCircle](https://github.com/lnsomniak/full-circle) - Intelligent Music Discovery Engine
> ML powered personalized music recommendation system that understands your taste

**The Problem:** Spotify's recommendations are trained on everybody. FullCircle learns *your* specific taste - the energy you look for, the moods you gravitate toward, how you jump between completely different genres.

**Key Features:**
- ü§ñ XGBoost ML model with TF-IDF for prediction confidence (85% accuracy)
- üéØ Weighted similarity using rare tag IDF scores
- üìä Cosine similarity calculations for artist matching
- üîÆ Explore Mode - Hides artists you already know for pure discovery
- üí° Explainable AI - See WHY each artist was recommended
- üëçüëé Feedback collection system for model retraining
- üé® Personalized loading screen with your album artwork

**Tech Stack:**
- **Frontend:** Next.js, React, TypeScript, Tailwind CSS
- **Backend:** FastAPI (Python), Spotify Web API, Last.fm API
- **ML/Data:** XGBoost, TF-IDF, scikit-learn, NumPy, SciPy
- **Database:** Supabase (PostgreSQL)
- **Deployment:** Vercel + Render

**Live Demo:** https://full-circle-theta.vercel.app
